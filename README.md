# Coffee Sales Analytics Dashboard

Project Overview
This project analyzes coffee shop sales data and predicts future sales using machine learning.

## Technologies Used

Python
MySQL
Pandas
Scikit-Learn
Streamlit

## Features

Sales data analysis
Interactive dashboard
7-day sales forecasting
Revenue insights

## How to Run

Install dependencies

pip install -r requirements.txt

Train the model

python src/ml_model.py

Run dashboard

streamlit run dashboard/app.py
